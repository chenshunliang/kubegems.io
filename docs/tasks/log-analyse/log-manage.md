## 日志管理
  查询的日志结果可以已图表方式来呈现,点击`图表`,可以看到日志时序的图标展示  

![pic](./log-query-graph.png)

  日志结果窗口上可以进行如下功能调整
  - `限制`输入框可以调整显示日志数量
  - `时间`开关可以显示日志采集的时间点
  - `级别`选择按钮可以调整显示日志的级别

![pic](./log-query-manage.png)

  日志结果可以通过浏览器下载,点击`下载日志`

![pic](./log-query-download.png)
