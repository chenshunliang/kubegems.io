存储卷快照

快照​

存储卷快照 是存储卷在某一时刻的完整备份，您可以通过存储卷快照恢复您的存储卷。

通常对于重要数据，都会选择定期对数据进行备份；CSI的Capacity提供了是否支持卷快照; kubegems 采用了外部快照的方案;

快照恢复​

您可以从现有的快照创建一个新的 存储卷
